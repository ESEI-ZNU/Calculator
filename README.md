# Project: Number Conversion

## Опис проєкту Dev2

Проєкт призначений для перетворення цілих чисел між десятковою та двійковою системами числення.

Програма:
- перетворює десяткове число у двійковий рядок;
- перетворює двійковий рядок у десяткове число;
- підтримує додатні, від’ємні числа та нуль;
- перевіряє коректність вхідних даних;
- демонструє роботу методів у `Number`.

---

## Архітектура проєкту

Проєкт побудований як один утилітний клас `Number`, який містить лише статичні методи.

### Клас `Number`

**Модифікатор:** `public`

**Призначення:**  
Містить методи для перетворення чисел між системами числення та точку входу в програму.

**Члени класу:**

#### Поля
Поля відсутні.

#### Конструктори
Конструктори відсутні, оскільки клас використовується як утилітний.

#### Методи

##### `public static String toBinaryString(int number)`
Перетворює ціле десяткове число у двійковий рядок.

**Параметри:**
- `number` — число у десятковій системі числення.

**Результат:**
- двійкове рядкове представлення числа;
- для від’ємних чисел додається знак `-`;
- для `0` повертається `"0"`.

##### `public static int fromBinaryString(String binary)`
Перетворює рядок з двійковим записом у десяткове ціле число.

**Параметри:**
- `binary` — рядок у двійковій системі числення.

**Результат:**
- десяткове значення, яке відповідає переданому рядку.

**Виключення:**
- `IllegalArgumentException` — якщо рядок `null`, порожній або містить недопустимі символи.

##### `public static void main(String[] args)`
Демонструє роботу методів класу.

**Параметри:**
- `args` — аргументи командного рядка.

**Результат:**
- вивід прикладів роботи в консоль.

## Логіка роботи програми

### Перетворення десяткового числа у двійкове
1. Перевіряється, чи число дорівнює нулю.
2. Визначається знак числа.
3. Для від’ємних чисел береться модуль.
4. Число послідовно ділиться на 2.
5. Остачі від ділення формують двійковий запис.
6. Якщо число було від’ємним, перед результатом додається `-`.

### Перетворення двійкового рядка у десяткове
1. Перевіряється, чи рядок не є `null` або порожнім.
2. Визначається наявність знака `-`.
3. Перевіряється, що рядок містить лише символи `0` та `1`.
4. Значення обчислюється від молодшого біта до старшого.
5. За потреби результат робиться від’ємним.

---

## Приклади використання

```java
toBinaryString(47)   // "101111"
toBinaryString(-47)  // "-101111"
toBinaryString(0)    // "0"

fromBinaryString("101111")   // 47
fromBinaryString("-101111")  // -47
fromBinaryString("0")        // 0
```

---

## Критерії готовності

Проєкт вважається готовим, якщо:

- коректно перетворює додатні числа у двійковий запис;
- коректно перетворює від’ємні числа у двійковий запис;
- правильно обробляє `0`;
- коректно виконує зворотне перетворення;
- виявляє помилки введення та генерує `IllegalArgumentException`;
- приклади з `Number()` виконуються без помилок.

У проєкті реалізовано двостороннє перетворення чисел між десятковою та двійковою системами числення. Під час роботи застосовано статичні методи, рядкову обробку, цикли, умовні оператори та перевірку коректності вхідних даних.
